# Archive

This folder contains historical experiment iterations and old experiment artifacts.

## What Is Archived Here

- old Section 4.2 / 4.3 / 4.4 / 4.6 experiment directories
- earlier detection-accuracy implementations
- older result JSON/PNG artifacts
- older standalone experiment scripts that are no longer the main code path

## When To Open This Folder

Only open this folder if you want to:

- trace the evolution of an experiment
- compare an old implementation against the latest pipeline
- recover an older figure or intermediate artifact

For normal reading, use `paper_experiments/latest_release/` instead.
